# undefinedBot
This bot has been made for the undefined discord server.
https://discordapp.com/invite/XDdZDZk

## Get Started
Install npm dependencies
```
$ npm install
```

Create key.js file to store you api key
```
module.exports = {
    discordKey: "yourApiKey",
    giphyKey: "yourApiKey",
    googleKey: "youApiKey"
};
```

Start bot with
```
$ node app.js
```
